# Upstream License and Terms Pointers

FlatLands is a derived dataset. The FlatLands release notice in `LICENSE` does not replace or weaken the upstream source dataset terms.

Users must review and comply with the current terms for each source dataset before using FlatLands observations derived from that source.

## Practical Notes

- Some upstream datasets restrict redistribution, commercial use, or require separate registration.
- FlatLands users should treat per-observation `metadata.json` source fields as the authority for which upstream terms apply.
- Do not assume that a permissive software repository license applies to all underlying source data unless the upstream project explicitly says so.
